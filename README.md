# stock

## 需求
* Redis
* PHP 7.1
* MySQL
* NPM
* Vue.js
* Composer
* FB Developer 

## 安裝
* 下載專案              git clone
* 安裝php套件           composer install
* 安裝js套件            npm install
* Build js             npm run dev
* 複製環境檔             cp .env.example .env
* 產生laravel 加密參數   php artisan key:generate
* 執行redis             redis-server

## file .env 設定
* FB_ID=
* FB_SECRET=
* FB_CALLBACK=
